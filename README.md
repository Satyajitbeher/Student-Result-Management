# Student Result Management System

## Description
A simple Student Result Management System built in **Java** with **Swing GUI**.  
Allows users to **add, view, and search student records**.  
Student data is stored in a **file (students.txt)** for persistence.

## Features
- Add new student with roll number, name, and marks
- View all student records in a table
- Search student by roll number
- Simple and user-friendly GUI

## Technologies Used
- Java (OOP concepts)
- Java Swing (GUI)
- File handling for data storage
